## üìñ How to Use

### Creating a Detailed Profile

**Step 1: Add New Contact**
1. Click "+ Add Contact" button
2. See the tabbed modal interface

**Step 2: Fill Out Each Tab**

**Tab 1 - Basic Info** (Required)
- Enter vendor/business name (required)
- Add contact person details
- Email is required for all contacts

**Tab 2 - Business**
- Select category and segment
- Set initial status
- Add company size and revenue info
- Note lead source and referral

**Tab 3 - Contact Details**
- Add social media profiles
- Complete address information
- These help with research and outreach

**Tab 4 - Deal Info**
- Track where they are in sales pipeline
- Set deal value and probability
- Mark as decision maker
- Add budget information

**Tab 5 - Notes & Tags**
- Public notes: Info you can share with client
- Internal notes: Private team notes (yellow highlighted)
- Next steps: Action items
- Select relevant tags

**Step 3: Save**
- Click "Save Contact"
- All information saved to browser
- Instantly searchable and filterable

### Editing Existing Profiles

1. View any contact detail page
2. Click "Edit Profile" button
3. All current information pre-populated
4. Make changes across any tab
5. Save updates

### Using Tags Effectively

**Best Practices:**
- **Hot Lead** - High interest, quick follow-up needed
- **High Priority** - Important accounts to close
- **Decision Maker** - Contact has authority to sign
- **Budget Approved** - Money allocated, ready to buy
- **Large Account** - High deal value potential
- **Needs Follow-up** - Action required
- **Referral** - Came from existing customer/network

**Tips:**
- Use multiple tags per contact
- Tags visible in contact table for quick scanning
- Filter by tags in future updates

### Tracking Deals Through Pipeline

**Deal Stages:**
1. **Prospecting** - Initial research/outreach
2. **Qualified** - Confirmed fit, has budget/need
3. **Proposal** - Sent formal proposal
4. **Negotiation** - Discussing terms
5. **Closed Won** - Deal signed!
6. **Closed Lost** - Didn't close (track why in notes)

**Deal Fields:**
- **Value**: Expected revenue from this contact
- **Probability**: Your confidence level (0-100%)
- **Close Date**: When you expect to close
- **Authority**: Their decision-making power

### Organizing Notes

**Public Notes** (White background, blue border)
- Information you can share with the contact
- General observations
- Meeting summaries
- Safe for client eyes

**Internal Notes** (Yellow background, orange border)
- Private team information
- Competitive intelligence
- Concerns or red flags
- Strategy notes
- Never share with client!

**Next Steps**
- Specific action items
- What needs to happen before next contact
- Keeps everyone aligned

## üîÑ Workflow Examples

### Example 1: First Contact to Close

**Day 1 - Initial Outreach**
- Create basic profile (name, email, company)
- Add tag: "Hot Lead" if from warm intro
- Status: "Not Started"
- Send first email, log activity

**Day 3 - First Call**
- Edit profile, add phone number
- Complete business info tab
- Add social media during call
- Log call activity
- Status ‚Üí "In Progress"
- Set follow-up date

**Week 2 - Qualification Call**
- Update deal tab: Stage = "Qualified"
- Add deal value estimate
- Mark as decision maker (or not)
- Note budget info
- Add tags: "Budget Approved", "Decision Maker"
- Internal notes: Any concerns, competition info
- Public notes: Meeting summary
- Status ‚Üí "Responded"

**Week 3 - Proposal**
- Deal stage ‚Üí "Proposal"
- Probability: 60%
- Expected close date set
- Next steps: Follow up on proposal
- Log proposal sent activity

**Week 4 - Negotiation**
- Deal stage ‚Üí "Negotiation"
- Probability: 80%
- Internal notes: Negotiation points
- Update next steps

**Week 5 - Close**
- Deal stage ‚Üí "Closed Won"
- Status ‚Üí "Signed Up"
- Log win!
- Internal notes: What worked, learnings

### Example 2: Expo Lead Follow-up

**At Expo**
- Quick profile: Name, company, email
- Tag: "Expo" (create custom tag)
- Public notes: "Met at booth, interested in Q1 campaign"
- Photo of business card ‚Üí add details later

**Day After Expo**
- Complete profile from business card
- Add social media (research them)
- Send follow-up email
- Log activity

**3 Days Later**
- First call scheduled
- Research: Add company size, revenue (LinkedIn)
- Check their website, add notes
- Prepare talk track in internal notes

**First Call**
- Update during call: Authority level, budget info
- Log call immediately after
- Set follow-up for proposal
- Tag: "High Priority" if interested

## ‚ùì FAQs

**Q: Can I import my existing contacts?**
A: Yes! Use CSV import. Basic fields auto-map. Enhance profiles manually over time.

**Q: How do I bulk update tags?**
A: Currently one-by-one. Future update will include bulk actions.

**Q: Can I add custom tags?**
A: Not yet in UI, but you can edit the code to add more default tags.

**Q: What if I don't need all these fields?**
A: No problem! All fields optional except name and email. Use what's useful.

**Q: Can my team share profiles?**
A: Deploy online, each person has own data. Or export/import CSV to share.

**Q: Does this work on mobile?**
A: Yes! Fully responsive. Tabs work on mobile too.

**Q: Can I delete fields I don't use?**
A: Fields don't take space if empty. Leave blank if not needed.
